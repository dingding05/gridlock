# Gridlock - Android APK Builder

## How to get your APK in 5 minutes (free):

### Step 1 - Create GitHub repo
1. Go to https://github.com and sign in (or create free account)
2. Click the + button > New repository
3. Name it "gridlock", set to Public, click Create

### Step 2 - Upload these files
1. Click "uploading an existing file" on the repo page
2. Drag ALL files/folders from this zip into the upload area
   (keep the folder structure exactly as-is)
3. Click "Commit changes"

### Step 3 - Run the build
1. Click the "Actions" tab in your repo
2. Click "Build Gridlock APK" on the left
3. Click "Run workflow" > "Run workflow"
4. Wait ~5 minutes

### Step 4 - Download APK
1. Click the completed workflow run
2. Scroll down to "Artifacts"
3. Click "Gridlock-APK" to download
4. Unzip it - inside is app-release.apk

### Step 5 - Install on Android
1. Send the APK to your phone (email, WhatsApp, Google Drive, USB)
2. On your phone: Settings > Apps > Install unknown apps > allow your browser/files app
3. Tap the APK file > Install
